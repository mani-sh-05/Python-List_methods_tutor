# Python List Learning Platform

## Introduction

This is a simple Python program made for practicing Python Lists. Initially, the task was to create a list and allow the user to perform operations like append and remove on it.

Instead of only performing operations, I tried to make the program a little more interactive. The program acts like a small learning platform where users can learn different list methods and also try them on their own list.

## Features

* Create a list by taking input from the user.
* Learn different list methods such as:

  * append()
  * remove()
  * insert()
  * pop()
  * clear()
  * sort()
  * reverse()
  * count()
  * index()
  * extend()
* Displays:

  * Definition of the method
  * Syntax of the method
  * Example code and output
* Allows the user to try the method on their own list.
* Shows both the initial list and the updated list after modification.
* Menu driven interface for easy navigation.

## Working

1. User enters the size of the list.
2. User enters elements of the list.
3. Program displays a menu of available list methods.
4. After selecting a method, the program shows:

   * Definition
   * Syntax
   * Example
5. User can choose to try the method on the current list.
6. The updated result is displayed.
7. User can continue learning more methods or exit the program.

## Concepts Used

* Lists
* Nested Dictionaries
* Functions
* Loops
* Conditional Statements
* User Input
* List Methods

## Purpose

The main purpose of this project is to practice Python list operations in an interactive way. It can also help beginners understand how different list methods work by allowing them to experiment with their own data.
